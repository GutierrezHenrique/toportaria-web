# ToPortaria — Web (React + Vite)

Painel administrativo: dashboard, visitantes, encomendas, moradores, unidades, logs, usuários.

## Stack
- Vite + React 18 + TypeScript
- Tailwind CSS (design system dark, accent coral)
- React Query + Zustand + React Router
- Recharts, Lucide, Sonner

## Dev

```bash
cp .env.example .env
npm install
npm run dev
```

Requer backend rodando em `http://localhost:3333`.

Login demo: `admin@toportaria.com` / `123456`.
